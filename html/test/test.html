Tests à effectuer

Q1 - Pays avec voisins sur différents continents

Affiche les pays ayant des voisins situés sur un autre continent.

Q2 - Pays avec le plus de voisins

Affiche les pays avec le plus grand nombre de pays frontaliers.

Q3 - Pays sans voisins

Montre les pays n'ayant aucune frontière terrestre avec d'autres pays.

Q4 - Pays avec le plus de langues

Affiche les pays avec le plus grand nombre de langues officielles/parlées.

Q5 - Pays avec voisins parlant une langue commune

Liste les pays ayant au moins un voisin partageant une langue commune.

Q6 - Pays sans voisins avec une monnaie commune

Affiche les pays dont aucun voisin n'utilise la même monnaie.

Q7 - Pays triés par densité de population décroissante

Trie et affiche les pays du plus densément peuplé au moins dense.

Q8 - Pays avec plusieurs Top Level Domains

Affiche les pays disposant de plusieurs extensions de domaine (.fr, .gp, etc).
